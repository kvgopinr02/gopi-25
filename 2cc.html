Temperature Conversion Program

Algorithm

  1. Start the program.
  2. Display options for conversion:
    • 1. Celsius to Fahrenheit
    • 2. Fahrenheit to Celsius
  3. Read user choice.
  4. If the choice is 1:
    • Prompt user for temperature in Celsius.
    • Read the Celsius value.
    • Convert to Fahrenheit using the formula: fahrenheit = (9.0 / 5.0) * celsius + 32.0;
    • Display the converted Fahrenheit temperature.
  5. If the choice is 2:
    • Prompt user for temperature in Fahrenheit.
    • Read the Fahrenheit value.
    • Convert to Celsius using the formula: celsius = (5.0 / 9.0) * (fahrenheit - 32.0);
    • Display the converted Celsius temperature.
  6. If the choice is invalid, display an error message.
  7. End the program.
C Code:

#include <stdio.h>

int main() {
    int choice;
    double celsius, fahrenheit;

    printf("Temperature Conversion:\\n");
    printf("1. Celsius to Fahrenheit\\n");
    printf("2. Fahrenheit to Celsius\\n");
    printf("Choice: ");
    scanf("%d", &choice);

    if (choice == 1) {
        printf("Enter Temperature in Celsius: ");
        scanf("%lf", &celsius);
        fahrenheit = 9.0 / 5.0 * celsius + 32.0;
        printf("Fahrenheit Temperature: %.2lf\\n", fahrenheit);
    } else if (choice == 2) {
        printf("Enter Temperature in Fahrenheit: ");
        scanf("%lf", &fahrenheit);
        celsius = (5.0 / 9.0) * (fahrenheit - 32.0);
        printf("Celsius Temperature: %.2lf\\n", celsius);
    } else {
        printf("Invalid choice\\n");
    }

    return 0;
}
