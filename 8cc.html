Toggle Case of String

Algorithm

  1. Start the program.
  2. Declare a character array s to store the string.
  3. Prompt the user to input a string.
  4. Read the string using fgets.
  5. Iterate through each character of the string:
    • If the character is lowercase, convert it to uppercase.
    • If the character is uppercase, convert it to lowercase.
  6. Display the modified string.
  7. End the program.
C Code:

#include <stdio.h>

int main() {
    char s[100];
    int i;
    printf("Enter a string: ");
    fgets(s, sizeof(s), stdin);
    
    for (i = 0; s[i] != '\0'; i++) {
        if (s[i] >= 'a' && s[i] <= 'z') {
            s[i] = s[i] - 32;
        } else if (s[i] >= 'A' && s[i] <= 'Z') {
            s[i] = s[i] + 32;
        }
    }
    
    printf("Result: %s\\n", s);
}
